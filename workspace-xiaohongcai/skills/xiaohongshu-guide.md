# 小红书运营指南

## 平台特性
- 用户群体：18-35岁女性为主，一二线城市
- 内容形式：图文笔记、短视频
- 算法机制：CES评分（点赞+收藏+评论+转发+关注）

## 爆款公式

### 标题模板
1. **数字型**: "3个技巧让你..."
2. **痛点型**: "为什么你总是..."
3. **对比型**: "月薪3k vs 月薪30k的区别"
4. **悬念型**: "这个方法让我..."
5. **身份型**: "作为XX，我必须告诉你..."

### 正文结构
```
开头（痛点/共鸣）→ 中间（干货/步骤）→ 结尾（互动/引导）
```

### emoji使用
- 🌟 重点标记
- 💡 小贴士
- 🔥 热门/紧急
- ✅ 完成/正确
- ❌ 错误/避免
- 📌 收藏提醒

## 发布节奏
- 最佳时间：7-9点、12-14点、19-22点
- 频率：每天1-2篇
- 话题标签：3-5个精准标签

## 互动策略
- 发布后30分钟内回复所有评论
- 用小号在评论区提问引导讨论
- 定期做抽奖/福利活动
