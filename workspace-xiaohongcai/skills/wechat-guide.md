# 公众号运营指南

## 平台特性
- 用户群体：全年龄段，深度阅读偏好
- 内容形式：图文、长文、音频
- 传播机制：朋友圈分享、在看、收藏

## 爆款公式

### 标题类型
1. **观点型**: "我认为未来10年..."
2. **干货型**: "最全的XX指南"
3. **故事型**: "那个年薪百万的年轻人"
4. **热点型**: "从XX事件看..."
5. **反常识**: "别再XX了，真相是..."

### 文章结构
```
引言（钩子）→ 背景 → 核心论点 → 案例分析 → 总结 → CTA
```

### 排版规范
- 字号：14-16px正文，18-20px小标题
- 行距：1.5-1.75倍
- 段落：每段不超过3行
- 配色：主色+辅助色，不超过3种
- 配图：封面图900x383，正文图宽度一致

## 发布节奏
- 最佳时间：周二-周四 19:00-21:00
- 频率：每周2-3篇深度长文
- 系列化：打造专栏，培养阅读习惯

## 互动策略
- 文末设置投票/问答
- 精选留言并回复
- 定期做读者调研
