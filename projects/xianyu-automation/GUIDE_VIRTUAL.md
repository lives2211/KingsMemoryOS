## 🚀 快速开始

### 第一步：准备资料

1. **收集资料**（你有以下几种方式）
   - 自己整理的学习资料
   - 购买的课程/素材（确保可以转售）
   - 网上收集的公开资源

2. **上传到百度网盘**
   - 创建分享链接
   - 设置提取码（可选）
   - 测试链接是否有效

### 第二步：修改商品数据

编辑 `data/virtual_items.json`：

```json
{
  "id": "virtual_001",
  "title": "【自用出】Python编程全套教程",
  "price": 29,           // 修改价格
  "size": "15GB",        // 修改实际大小
  "includes": [          // 修改具体内容
    "你的内容1",
    "你的内容2"
  ],
  "reason": "你的出售原因"
}
```

### 第三步：生成内容

```bash
cd /home/fengxueda/.openclaw/workspace/projects/xianyu-automation

# 生成标题和描述
python3 scripts/xianyu_content_virtual.py \
  --input data/virtual_items.json \
  --output data/generated_virtual.json
```

### 第四步：发布到闲鱼

**手动发布（推荐新手）：**
1. 打开 `data/generated_virtual.json`
2. 复制生成的标题和描述
3. 粘贴到闲鱼发布页面
4. 上传一张封面图（可以用Canva做）
5. 设置自动发货（闲鱼Pro版功能）

**自动发布（需API权限）：**
```bash
python3 scripts/xianyu_publish.py \
  --items data/virtual_items.json \
  --dry-run  # 先测试
```

## 💡 虚拟资料运营技巧

### 1. 选品策略

**热门品类（按转化率排序）：**
1. 考试资料（考研/公考/四六级）
2. 软件工具（Adobe/Office/编程）
3. 自媒体课程（小红书/抖音/公众号）
4. 设计素材（PSD/字体/样机）
5. 学习教程（编程/英语/技能）

### 2. 定价技巧

```
心理价位：
• ¥9.9 - 引流款，薄利多销
• ¥19-29 - 标准款，主力产品
• ¥39-49 - 高价值款，全套资料
• ¥99+ - 高端款，独家资源
```

### 3. 标题优化

**公式：** `[标签] + [核心卖点] + [关键词]`

**好标题示例：**
- ✅ 【自动发货】考研英语全套资料 真题+词汇+作文
- ✅ 【网盘秒发】10万+设计素材包 PSD+字体+样机
- ❌ 卖资料
- ❌ 教程便宜出

### 4. 描述要点

必须包含：
1. **资料介绍** - 是什么、多大、什么格式
2. **具体内容** - 列出1、2、3、4...
3. **使用说明** - 怎么获取、怎么使用
4. **售后说明** - 虚拟商品不退、链接失效补发

### 5. 自动发货设置

**闲鱼Pro版自动发货：**
1. 开通闲鱼Pro版（免费）
2. 设置自动回复关键词
3. 配置发货内容（网盘链接）

**关键词设置：**
- "在吗" → 自动回复购买说明
- "怎么买" → 自动回复下单流程
- "已拍" → 自动发送网盘链接

## ⚠️ 避坑指南

### 1. 版权问题
- ❌ 不要卖盗版课程（有法律风险）
- ✅ 可以卖自己整理的资料
- ✅ 可以卖公开资源合集

### 2. 违规词
闲鱼会屏蔽以下词，用替代词：
- ❌ "微信" → ✅ "薇信" 或 "VX"
- ❌ "QQ" → ✅ "球球" 或 "扣扣"
- ❌ "网盘" → ✅ "云盘" 或 "盘"
- ❌ "教程" → ✅ "资料" 或 "课程"

### 3. 账号安全
- 新账号每天擦亮不超过10个
- 不要频繁发布（每小时不超过5个）
- 不要直接留联系方式在描述里
