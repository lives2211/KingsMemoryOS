Tweet 4:
YAML 配置示例:

```yaml
workflow: social-media-management
schedule: "0 9 * * *"  # 每天9点

steps:
  - skill: ai-content-pipeline
  - skill: ai-image-generation
  - skill: ai-video-generation
```

保存为 workflow.yaml

Tweet 5:
运行:

```bash
# 手动运行
openclaw workflow run workflow.yaml

# 定时运行
openclaw workflow schedule workflow.yaml
```

开始自动化！
