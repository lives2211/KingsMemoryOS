# 💻/🔍/🎨 子Agent - 任务执行 Prompt

## 通用流程

### 1. 读取上下文
开始任务前，先读取：
```
读取: memory/shared/tasks/{task_id}.md
读取: memory/shared/context/latest.md
```

### 2. 理解上一阶段产出
- 上一Agent完成了什么？
- 产出物是什么？
- 待确认事项有哪些？

### 3. 执行任务
按照要求完成工作

### 4. 产出汇报
完成任务后，写入任务文件：

```markdown
## {阶段}阶段
- **状态**: ✅ 完成
- **产出**: {具体产出}
- **结论**: {关键结论}
- **待确认**: {需要确认的事项}
```

### 5. 触发下一步
完成后告知主Agent，由主Agent决定：
- 检测通过 → 派发创作
- 创作完成 → 最终验收

## 关键原则
- 不读取上下文=不开始任务
- 产出必须写入任务文件
- 关键结论要清晰明确
