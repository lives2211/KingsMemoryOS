# 共享记忆系统 - Agent配置

## 核心机制
```
Agent完成任务 → memory_store(scope=global) → 其他Agent用memory_recall查询
```

## 各Agent System Prompt 添加

### 所有Agent通用
```
## 共享记忆
- 任务开始前: 用 memory_recall 查询相关上下文
- 任务完成后: 用 memory_store 存入关键产出 (scope="global")
```

### 主Agent专用额外
```
## 任务管理
- 创建任务: 写入 memory/shared/tasks/{task_id}.md
- 派发任务: 附上上一阶段的memory_recall结果
- 验收任务: 汇总所有memory_store的产出
```

## 使用示例

**编程Agent完成后:**
```python
memory_store(
    text="任务xxx产出: 实现了xxx功能，文件路径xxx",
    scope="global",
    category="fact",
    importance=0.8
)
```

**检测Agent开始时:**
```python
memory_recall(
    query="任务xxx 编程产出",
    scope="global"
)
```

## 关键原则
- 产出必须存入global scope
- 查询时用任务ID作为关键词
- category统一用"fact"
