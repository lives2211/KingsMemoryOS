# 🧠 Agent 定期反思机制

## 反思频率
- **小反思**：每次任务完成后（耗时<5分钟）
- **大反思**：每天一次（耗时约15分钟）

---

## 🎯 小反思（任务后）

每次完成重要任务后，快速过一遍：

```
1. 这次协作顺畅吗？
2. 有没有可以改进的地方？
3. 下次遇到类似问题，怎么处理更好？
```

**输出**：一句话总结，记录到当日 memory 文件

---

## 🧩 大反思（每日）

每天定时执行，全面复盘：

### 复盘清单

```
📊 今日任务回顾
- 完成了哪些任务？
- 哪些做得好？哪些做得不好？

🤝 协作评估
- 沟通效率如何？有没有废话太多？
- 什么时候该找队友没找？什么时候不该找找了？
- 主Agent协调得怎么样？

🛠 技能使用
- 有没有不会的技能应该去学？
- 有没有更好的工具/方法没用上？

🎯 改进计划
- 明天要改进的3件事
```

### 执行方式

**方式A：各自反思（推荐）**
```
每个Agent独立填写上面的复盘清单
→ 写入 memory/YYYY-MM-DD.md
→ 关键改进点同步到 COLLABORATE.md
```

**方式B：团队复盘**
```
主Agent发起：
"大家来复盘一下今天的工作"
各Agent依次发言（简洁版）
→ 主Agent汇总改进建议
→ 更新协作协议
```

---

## 📚 记忆沉淀

每次反思后，必须做：

| 反思内容 | 存到哪里 |
|----------|----------|
| 做得好的案例 | COLLABORATE.md 的"正确协作示例" |
| 踩过的坑 | COLLABORATE.md 的"自愈铁律" |
| 技能新发现 | SKILLS_SHARE.md |
| 重要经验 | MEMORY.md |

---

## ⚙️ 自动触发

使用 Cron Job 每天固定时间触发大反思：

```
时间：每天 21:00（晚上9点）
触发：主Agent在群里发起反思
```

---

> 🧬 **进化公式**  
> 反思 → 记录 → 改进 → 实践 → 再反思 = 越来越智能
