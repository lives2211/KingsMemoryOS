Tweet 4:
Step 3: 核心代码

```python
# src/index.py
from openclaw import Skill

class MySkill67Skill(Skill):
    def run(self, input_data):
        # Your logic here
        result = process(input_data)
        return result
```

保持代码简洁。

Tweet 5:
Step 4: 测试

```bash
# 本地测试
openclaw skill test my-skill-67

# 调试模式
openclaw skill test my-skill-67 --debug
```

确保所有测试通过。

Tweet 6:
Step 5: 发布到社区

```bash
# 提交到 ClawHub
openclaw skill publish my-skill-67

# 分享你的 Skill
openclaw skill share my-skill-67
```

社区会帮你 review。

Tweet 7:
最佳实践:

✅ 写清晰的文档
✅ 提供使用示例
✅ 处理错误情况
✅ 添加单元测试
✅ 保持代码整洁

❌ 不要硬编码 API keys
❌ 不要忽略错误处理
