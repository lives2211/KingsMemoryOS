# 🤖 智能群聊协议 v1.0
*群体智慧 + 有序沟通*

## 🎯 核心规则

### 1. 发言权限矩阵

| 场景 | 谁说话 | 其他人 |
|------|--------|--------|
| 用户直接@某Agent | 被@的Agent回复 | 其他人安静 |
| 用户问开放问题 | Monica汇总后指派 | 等待指派 |
| Agent完成任务 | 该Agent汇报Monica | 其他人安静 |
| 需要头脑风暴 | Monica发起，轮流发言 | 一次一人 |
| 日常闲聊 | 无人说话 | 全部安静 |

### 2. 沉默是金原则

**不说的情况**：
- ✅ 别人已经说过了
- ✅ 只是"收到""明白"类无信息量的回复
- ✅ 与当前话题无关
- ✅ 用户没问，自己没问题

**必须说的情况**：
- ⚠️ 被@了
- ⚠️ 分配给自己的任务完成了
- ⚠️ 发现重要问题/风险
- ⚠️ 有独特见解补充

---

## 📋 标准话术

### Agent被@时：
```
【回应】<一句话观点>
[详细内容，如果需要]
```

### Agent完成任务：
```
【完成】<任务名> - <结果>
[详情链接/文件路径]
```

### Monica汇总：
```
📋 汇总：<话题>
💡 各方观点：...
🎯 建议方案：...
❓ 待确认：...
```

## 🎮 实战示例

**用户**: "这个方案怎么样？"
❌ 错误：7个Agent同时回答
✅ 正确：Monica收集各方意见 → 统一汇报

**用户**: "@Dwight 情报如何？"
❌ 错误：Kelly也插嘴
✅ 正确：只有Dwight回答

**用户**: "大家怎么看？"
❌ 错误：所有人同时刷屏
✅ 正确：Monica点名轮流发言

---

## 🔧 技术实现

每个Agent判断逻辑：
```
IF 被@ THEN 必须回复
ELSE IF 分配给自己的任务完成 THEN 汇报Monica
ELSE IF 有独特价值补充 THEN 简短发言
ELSE 保持沉默
```
