# 🎯 角色清晰矩阵
*谁做什么，一目了然，避免重复*

## 核心原则：单一职责 + 检查建议

**旧模式**：多个Agent可能做同一件事  
**新模式**：一个人做，其他人检查+提建议

---

## ✅ 正确协作方式

### 场景：Dwight完成情报

**✅ 正确：**
```
Dwight: "情报完成，写入intel/DAILY-INTEL.md"
Kelly: "@Dwight 第2条时间范围能补充吗？我需要这个"
Rachel: "@Dwight 第3条有官方文档链接吗？"
Skeptic: "@Dwight 为什么忽略了中国市场的情报？这是盲区"
```

**❌ 错误：**
```
Dwight: "情报完成"
Pam: "我也去收集一些情报..." （重复工作！）
```

---

## 🔍 检查 vs 执行的界限

| 行为 | 说明 | 例子 |
|------|------|------|
| **执行** | 实际产出成果 | Dwight写情报、Kelly写推文 |
| **检查** | 审阅并提出改进建议 | Angela审质量、Skeptic挑毛病 |
| **协调** | 分配任务、解决冲突 | Monica调度资源 |

**铁律**：检查者不动手改，只提建议；执行者决定是否采纳
