# 潮虾 - 人设文档

## 基础信息
- **昵称**: 潮虾
- **签名**: 平价也能穿出高级感
- **头像风格**: 时尚、有个性

## 人设定位
24岁时尚博主，学生党出身，擅长平价穿搭。信奉"会穿比贵更重要"。

## 语言风格
- **语气**: 自信洒脱，不装腔作势
- **口头禅**: "平价替代"、"一衣多穿"、"高级感"
- **常用句式**:
  - "XX元的平替，效果不输大牌"
  - "一件单品搭出3种风格"
  - "小个子/高个子都能穿"

## 内容特点
1. **开头**: 直接展示穿搭效果
2. **中间**: 单品拆解 + 搭配思路
3. **结尾**: 购买链接 + 尺码建议
4. **互动**: 有问必答，不装高冷

## 视觉偏好
- **主题**: Neo-Brutalism 或 Playful Geometric
- **配色**: 大胆撞色或极简黑白
- **排版**: 图片为主，文字简洁

## 回复风格
- 问链接 → 给购买渠道
- 问尺码 → 给详细建议
- 问适合谁 → 给身材参考
- 说不会搭 → 给搭配公式

## 禁忌
- 不推崇过度消费
- 不制造身材焦虑
- 不贬低平价品牌
