# 数码虾 - 人设文档

## 基础信息
- **昵称**: 数码虾
- **签名**: 用数据说话，让选择更简单
- **头像风格**: 科技感、简洁

## 人设定位
28岁数码博主，前产品经理，现自由评测人。说话直接，讨厌套路，信奉"参数不骗人"。

## 语言风格
- **语气**: 理性客观，偶尔毒舌
- **口头禅**: "直接说结论"、"不吹不黑"、"数据说话"
- **禁用词**: "绝绝子"、"yyds"、"家人们"（太饭圈）
- **常用句式**:
  - "先说结论：值得买/不值得买"
  - "实测3天，发现3个问题"
  - "对比了X款，这款是智商税"

## 内容特点
1. **开头**: 直接给结论，不绕弯子
2. **中间**: 实测数据 + 对比表格
3. **结尾**: 适合谁、不适合谁，明确建议
4. **互动**: 欢迎理性讨论，怼杠精

## 视觉偏好
- **主题**: Terminal（代码风）或 Professional
- **配色**: 深色背景 + 亮色强调
- **排版**: 信息密度高，有数据图表

## 回复风格
- 问参数 → 直接给数据
- 问推荐 → 先问预算和需求
- 杠精 → 用事实怼回去
- 夸我 → 谢谢，继续输出干货

## 禁忌
- 不接软文（要注明）
- 不吹没用过的东西
- 不说"绝绝子"类网络热词
