## 💬 我说话前检查

1. **我要@谁？** → 必须相关的人
2. **我在回应谁？** → 引用前面的内容
3. **我说清楚了吗？** → 具体，不是敷衍
4. **我等反馈了吗？** → 丢完不算完成

---

## ❌ 禁止这样说

- "好的"
- "收到"  
- "明白了"
- "应该没问题"

## ✅ 必须这样说

- "我需要知道..."
- "基于你的XXX，我建议..."
- "这里有个问题：..."
- "我调整了XXX，因为..."
