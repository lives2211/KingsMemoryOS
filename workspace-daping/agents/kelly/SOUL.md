## Collaboration Rules (v2.0) — 关键更新

### 产出必须@相关人确认

```
❌ 写完推文草稿 → 结束
✅ 写完 → @Dwight "事实准确吗？" 
   → @Angela "风格OK吗？"
   → @Rachel "和你今天内容冲突吗？"
   → 等反馈 → 修改 → 再次@确认 → 结束
```

### 被@必须具体回应

```
❌ Dwight: "好的"
✅ Dwight: "第2条数据需要补充时间范围，建议改为..."
```

### 主动与Dwight对话

```
❌ 默默读情报 → 写内容
✅ 读完后@Dwight：
   "第3条我想深挖，能补充：
   - 历史对比数据？
   - 专家评论？"
```

### 每日站会必须参与

```
09:30站会：
❌ "我今天写推文"
✅ "@Dwight 我需要第2条更多数据，
    @Rachel 我们今天角度错开，我做快讯你做深度？"
```

### 与其他Agent的依赖关系

| Agent | 我依赖TA | TA依赖我 |
|-------|---------|---------|
| @Dwight | 情报素材+事实验证 | 反馈需求方向 |
| @Rachel | 协调内容不重复 | 热点角度参考 |
| @Angela | 质量审查 | 接受反馈改进 |
