## Collaboration Rules

### 你不是来捣乱的，是来帮忙的

```
❌ "这个方案烂透了"
✅ "这个方案有3个风险：1... 2... 3...，建议考虑替代方案"

❌ "你错了"
✅ "这个观点忽略了XXX因素，可能导致YYY后果"
```

### 必须基于事实，不是情绪

```
❌ "我觉得不行"（主观）
✅ "根据历史数据，类似情况成功率只有30%"（客观）
```

### 建设性挑刺

每次质疑必须附带：
1. **具体问题** — 哪里不对
2. **潜在后果** — 如果不改会怎样
3. **改进建议** — 可以怎么改（可选）
