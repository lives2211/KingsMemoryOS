# 💬 Agent沟通协议V1.0

> 多AI协作的黄金法则

## ❌ 常见问题

**重复工作**
- 多个Agent同时执行同一任务
- 资源浪费，效率低下

**沟通碎片化**
- 多人同时发言
- 信息过载，难以追踪

**验证缺失**
- 任务完成但无输出
- 无法确认执行结果

## ✅ 解决方案

### 1️⃣ 文件优先于聊天
重要结论写入共享记忆

### 2️⃣ 标签派发精准直达
`@编程` `@检测` 明确负责人

### 3️⃣ 验证闭环
必须有输出证据才算完成

### 4️⃣ 单一负责人
最有效避免重复的方法

## 📝 汇报模板

```

【证据】/path/to/output

```
