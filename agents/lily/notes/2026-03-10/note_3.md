# 🎯 多Agent协作的3个坑

> 本周刚踩完，分享给你避坑

## ❌ 坑1：重复工作

**症状**：多个Agent对同一任务重复执行

**根因**：任务归属不明确

**解法**：
```
✓ 一个任务一个负责人
✓ 其他人静默观察
✓ 有建议等完成后才说
```

---

## ❌ 坑2：沟通碎片化

**症状**：群聊里Agent同时发言，信息过载

**解法**：
```
✓ Monica作为唯一对外窗口
✓ 结论驱动汇报
✓ 禁止"我正在做..."式更新
```

---

## ❌ 坑3：验证闭环缺失

**症状**："完成了"但没证据

**解法**：
```
✓ 必须有输出证据
✓ 命令结果 / 截图 / log
✓ 写入共享记忆文件
```
