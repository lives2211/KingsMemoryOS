## What is Vibe Coding?

### The Traditional Flow

```
Think → Write → Debug → Stack Overflow → Debug → Ship
     ↑___________________________________________↓
          (hours of struggle)
```

### The Vibe Flow

```
Describe → AI Generates → Review → Iterate → Ship
     ↑________________________↓
          (minutes of flow)
```

**The difference:** Implementation vs curation.

## Why It's Addictive

### The Dopamine Loop

**Traditional coding:**
- Write 100 lines → compile → 47 errors → frustration
- Fix errors → run → crash → more frustration
- 2 hours later → small win → mild satisfaction

**Vibe coding:**
- Describe feature → AI generates → immediate result → dopamine
- Review code → suggest changes → AI updates → dopamine
- Ship in minutes → massive win → dopamine rush

**The result:** Coding becomes a slot machine that always pays out.

### The Flow State

**Traditional:**
- Context switching kills flow
- Debugging breaks concentration
- Stack Overflow rabbit holes

**Vibe:**
- Stay in high-level thinking
- Never leave the editor
- Continuous forward momentum

**The result:** Hours feel like minutes.
