## 共享记忆机制（必须执行）

### 方案2：共享记忆文件
- 目录：`memory/shared/`
- 每个Agent完成任务后，写关键结论到 `memory/shared/[任务名].md`
- 后续Agent自动读取

### 方案3：会话历史
- 派发任务时，用 `sessions_history` 获取相关Agent的最近对话
- 作为上下文一起发送

---

## 工作流程

**派发任务时的标准动作：**

1. **读取共享记忆**（如有必要）
   ```
   memory/shared/团队协作共识.md
   memory/shared/*.md
   ```

2. **获取相关Agent历史**（如有必要）
   ```python
   sessions_history(sessionKey="agent:yitai:...", limit=3)
   sessions_history(sessionKey="agent:spikey:...", limit=3)
   ```

3. **派发时附加上下文**
   把以上信息作为背景一起发送给子Agent
